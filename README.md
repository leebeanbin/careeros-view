# careeros-view (테스트용 저장소)

> [!IMPORTANT]
> 본 저장소는 클로닝 기법을 활용하여 웹 애플리케이션을 더 효율적이고 완성도 높게 구축하기 전에, 사전 기능 검증 및 기술 검토를 목적으로 생성한 **테스트 및 프로토타이핑용 저장소**입니다.

## 📌 프로젝트 소개
이 프로젝트는 웹 사이트(Linear.app 스타일)의 디자인 토큰, 레이아웃 구조, 컴포넌트 생태계를 분석하고 Next.js 환경으로 복제하는 기술을 검증하기 위한 테스트베드입니다. 
사전에 추출한 컴포넌트 명세서와 레이아웃 아키텍처를 바탕으로 설계되었으며, 이후 고도화된 웹 애플리케이션 개발을 진행하기 전 기반 코드로 활용됩니다.

## 🛠️ 기술 스택
- **Framework:** Next.js 16 (App Router, React 19, TypeScript strict)
- **Styling:** Tailwind CSS v4 (oklch 디자인 토큰 활용)
- **Icons:** Lucide React & 커스텀 SVG 컴포넌트
- **Testing/QA:** Playwright (스크린샷 및 레이아웃 감사 기능)

## 📂 주요 검증 항목
1. **디자인 토큰 이식**: 다크 모드 테마 맵핑 및 oklch 컬러 팔레트 적용
2. **반응형 컴포넌트 구현**: 모바일-퍼스트 레이아웃 구조 및 마이크로 인터랙션
3. **다중 라우팅 검증**: 워크스페이스 대시보드 구조 및 공용/인증 서브페이지 라우팅
4. **자동화 검사**: Playwright 기반의 라우트별 렌더링 감사 및 스크린샷 캡처 테스트

## ⚙️ 실행 및 빌드 명령어
패키지 매니저로 `npm`을 사용합니다.

```bash
npm install      # 의존성 설치
npm run dev      # 개발 서버 실행 (localhost:3000)
npm run build    # 프로덕션 빌드 테스트
npm run lint     # 코드 린트 검사
npm run typecheck # 타입 검사
npm run check    # 린트 + 타입검사 + 빌드 종합 검사
```
